# 手动打包方案

## 问题原因
electron-packager 需要从网络下载 Electron 运行时，但网络连接不稳定导致失败。

## 解决方案

### 方案A：使用国内镜像（推荐先尝试）
双击运行 `使用国内镜像打包.bat`

### 方案B：手动下载 Electron
1. 访问 https://npmmirror.com/mirrors/electron/
2. 下载 `electron-v28.3.3-win32-x64.zip`
3. 解压到 `electron/electron-cache/` 目录
4. 运行 `pack-offline.js`

### 方案C：使用已安装的 Electron（最简单）
直接复制已安装的electron：

```batch
cd "f:/agent 视频/ai-video-editor/electron"

:: 创建输出目录
mkdir release\AI视频编辑器-win32-x64

:: 复制electron
xcopy /E /I node_modules\electron\dist release\AI视频编辑器-win32-x64

:: 复制应用文件
xcopy /E /I dist release\AI视频编辑器-win32-x64\resources\app
xcopy /E /I ..\client\dist release\AI视频编辑器-win32-x64\resources\app\app

:: 重命名exe
ren release\AI视频编辑器-win32-x64\electron.exe "AI视频编辑器.exe"
```

## 打包后的文件结构
```
release/
└── AI视频编辑器-win32-x64/
    ├── AI视频编辑器.exe    <- 主程序
    ├── resources/
    │   └── app/            <- 应用代码
    │       ├── dist/       <- Electron主进程
    │       └── app/        <- 前端页面
    └── ...其他electron文件
```

## 运行方式
双击 `AI视频编辑器.exe` 即可运行
