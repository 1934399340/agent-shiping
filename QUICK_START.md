# AI视频编辑器 - 快速使用指南

## 🚀 快速启动

### 方式一：一键启动（推荐）
```bash
# 双击运行
start.bat
```
这会自动启动后端和前端服务，并打开浏览器。

### 方式二：手动启动
```bash
# 终端1：启动后端（端口3001）
cd server && pnpm dev

# 终端2：启动前端（端口3000）
cd client && npx vite --port 3000 --host

# 浏览器访问：http://localhost:3000
```

## 🎬 基本使用

### 1. 搜索视频素材
1. 点击左侧"素材搜索"标签
2. 输入关键词（如：city, nature, people）
3. 选择素材源（推荐：Mixkit）
4. 点击搜索按钮
5. 点击素材上的"+"添加到时间线

### 2. 导入本地素材
1. 点击左侧"媒体库"标签
2. 拖拽文件到上传区域
   - 支持视频：mp4, webm, mov, avi
   - 支持音频：mp3, wav, ogg, m4a
   - 支持图片：jpg, png, gif, webp
3. 点击素材上的"+"添加到时间线

### 3. 添加文字
1. 点击右侧"文字"标签
2. 点击"添加文字片段"
3. 设置文字内容、字体、颜色等
4. 文字会显示在预览画面中

### 4. AI自动剪辑
1. 点击右侧"AI剪辑"标签
2. 选择预设模板或输入自定义脚本
3. 点击"开始AI自动剪辑"
4. 等待素材搜索和编排完成
5. 结果自动添加到时间线

## ⚠️ 常见问题

### Q: 搜索不到视频？
A: 确保后端服务已启动（端口3001）。运行 `start.bat` 自动启动。

### Q: 图片导入后看不到？
A: 已修复。图片会显示在预览画面中，作为静态视频片段处理。

### Q: 文字不显示？
A: 已修复。文字现在会叠加显示在预览画面中。

### Q: 网页太卡？
A: 建议打包成桌面应用。运行 `build-electron.bat`。

### Q: 端口被占用？
A: 修改启动命令中的端口号：
```bash
# 前端改用其他端口
cd client && npx vite --port 3000 --host

# 后端改用其他端口
cd server && PORT=3002 pnpm dev
```
