## 使用方法

### 方式一：开发模式（推荐）
```bash
# 双击运行 start.bat
# 或手动执行：
cd server && pnpm dev
cd client && npx vite --port 3000 --host
```

### 方式二：打包桌面应用
```bash
cd electron

# 安装依赖
pnpm install

# 打包（使用electron-packager）
pnpm exec electron-packager . "AI视频编辑器" --platform=win32 --arch=x64 --out=release-build --overwrite

# 或使用electron-builder
pnpm exec electron-builder --win --x64
```

## 输出文件
打包完成后，exe文件将位于：
- `electron/release-build/AI视频编辑器-win32-x64/AI视频编辑器.exe`
- 或 `electron/release/AI视频编辑器-0.1.0-x64.exe`

## 注意事项
1. 确保client和server都已构建（dist目录存在）
2. 打包过程可能需要几分钟
3. 首次运行需要安装依赖
